NES [V1.2]:

Hola, para la versión 1.2 he solucionado algunos errores y ahora se puede escribir los comandos en diferentes formas como por ejemplo; Hola, hola o HOLA. También he incorporado a última hora un comando que le veo futuro ya que es un apartado para juegos en el que intentare traer un juego nuevo en cada actualización. De momento el único juego disponible que hay es el ‘Piedra, papel y tijeras’. Los nuevos comandos de esta versión son; ‘Activar modo calculadora’ (Con el que podrás obviamente hacer cálculos), ‘Random’ (Que te ofrecerá un número aleatoriamente) y por último ‘Jugar’ (Como ya he mencionado es un conjunto de juegos). Si quieres estar mas informado de cuando sacaré las siguientes versiones de N.E.S, te recomiendo que visites de vez en cuando mi Twitter. Y con todo esto explicado solo me queda decir que disfrutes de N.E.S totalmente gratis y seguro, y estás invitado a comunicarme cualquier error o recomendación desde mi Twitter o la página oficial de Github de este proyecto. Para abrir este proyecto necesitas tener Python descargado esto lo puedes hacer desde la página oficial de Python.
